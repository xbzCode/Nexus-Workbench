# 设计系统

简历模板的设计规范，确保视觉一致性和专业感。

## 核心原则

- **留白至上**：内容与边距、板块与板块之间保持充裕空间
- **线条锚定**：通过细线和分隔线建立视觉层次
- **色彩克制**：大面积黑白灰 + 小面积主题色点缀
- **信息密度**：重要信息突出，次要信息弱化，避免信息过载

## 主题色方案

通过 CSS 变量 `--accent` 控制，一处修改全局生效：

```css
:root {
  --accent: #475569;           /* 主题色 */
  --accent-light: rgba(71, 85, 105, 0.08);   /* 浅底色（标签、照片边框） */
  --accent-medium: rgba(71, 85, 105, 0.15);   /* 中底色（标题分隔线） */
}
```

| 主题色 | --accent | --accent-light | --accent-medium |
|--------|----------|----------------|-----------------|
| slate | `#475569` | `rgba(71,85,105,0.08)` | `rgba(71,85,105,0.15)` |
