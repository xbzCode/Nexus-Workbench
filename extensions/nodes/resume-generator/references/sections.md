# 简历板块定义

每个板块的结构、字段说明和展示规则。

## 1. 头部信息（必要）

| 字段 | 必要 | 说明 |
|------|------|------|
| name | ✅ | 姓名，大号加粗显示 |
| title | ⬜ | 求职意向/职位标题 |
| phone | ⬜ | 手机号码 |
| email | ⬜ | 邮箱地址 |
| location | ⬜ | 所在城市 |
| website | ⬜ | 个人网站/博客 |
| github | ⬜ | GitHub 链接 |
| linkedin | ⬜ | LinkedIn 链接 |
| photo | ⬜ | 照片 URL 或 base64（`--no-photo` 时隐藏） |

展示规则：
- 姓名居中显示，下方紧跟求职意向
- 联系方式以图标+文字形式一行排列，用 `·` 分隔
- 照片在右侧显示（圆形裁剪），默认显示占位区域

## 2. 工作经历（必要，至少一项）

| 字段 | 必要 | 说明 |
|------|------|------|
| company | ✅ | 公司名称 |
| position | ✅ | 职位名称 |
| startDate | ✅ | 开始时间（YYYY.MM） |
| endDate | ✅ | 结束时间（YYYY.MM 或"至今"） |
| description | ⬜ | 职责描述（支持多条，每条以 `•` 开头） |
| tags | ⬜ | 技能标签数组 |

展示规则：
- 按时间倒序排列（最近经历在前）
- 公司名+职位在同一行，公司名加粗，时间右对齐
- 描述以列表形式展示，每条前加 `•`

## 3. 教育背景（必要，至少一项）

| 字段 | 必要 | 说明 |
|------|------|------|
| school | ✅ | 学校名称 |
| major | ✅ | 专业 |
| degree | ✅ | 学历（本科/硕士/博士/大专） |
| startDate | ✅ | 开始时间（YYYY.MM） |
| endDate | ✅ | 结束时间（YYYY.MM） |
| gpa | ⬜ | GPA/排名 |
| honors | ⬜ | 在校荣誉数组 |
| courses | ⬜ | 核心课程数组 |

展示规则：
- 按时间倒序排列
- 学校名+学历在同一行，学校名加粗，时间右对齐
- 专业在第二行，GPA/荣誉在第三行

## 4. 专业技能（可选）

| 字段 | 必要 | 说明 |
|------|------|------|
| category | ✅ | 技能分类名（如"编程语言"、"框架"） |
| skills | ✅ | 技能名称数组 |

展示规则：
- 按分类分组，分类名加粗
- 同一分类下的技能用 `·` 分隔，一行展示

## 5. 项目经验（可选）

| 字段 | 必要 | 说明 |
|------|------|------|
| name | ✅ | 项目名称 |
| role | ⬜ | 担任角色 |
| startDate | ⬜ | 开始时间 |
| endDate | ⬜ | 结束时间 |
| description | ⬜ | 项目描述 |
| highlights | ⬜ | 关键成果数组 |
| tags | ⬜ | 技术标签数组 |

展示规则：
- 按时间倒序排列
- 项目名+角色同行，时间右对齐
- 描述在前，成果列表在后

## 6. 自我评价（可选）

| 字段 | 必要 | 说明 |
|------|------|------|
| summary | ✅ | 自我评价文本 |

展示规则：
- 不超过 3-4 句话
- 以段落形式展示，不加项目符号

## 7. 证书/荣誉（可选）

| 字段 | 必要 | 说明 |
|------|------|------|
| name | ✅ | 证书/荣誉名称 |
| issuer | ⬜ | 颁发机构 |
| date | ⬜ | 获得时间 |

展示规则：
- 按时间倒序排列
- 名称+颁发机构同行，时间右对齐

## 8. 语言能力（可选）

| 字段 | 必要 | 说明 |
|------|------|------|
| language | ✅ | 语言名称 |
| level | ✅ | 水平（如"母语"、"流利"、"熟练"、"基础"） |

展示规则：
- 语言+水平同行，用 `—` 连接
