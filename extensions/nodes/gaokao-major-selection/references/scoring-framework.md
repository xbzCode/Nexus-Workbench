# 评分框架

## 价值观 → 评分维度映射

Stage 1 的 5 个价值观选项和 7 个评分维度不是 1:1 对应的。按以下规则映射：

| 价值观（Stage 1.4） | 映射到评分维度 | 说明 |
|---------------------|---------------|------|
| 1. 个人兴趣匹配度 | **兴趣匹配度** | 直接映射，一一对应 |
| 2. 就业前景与薪资水平 | **就业前景** + **薪资水平** | 一对二，自定义权重时两个维度各享该排名权重的 50% |
| 3. 工作稳定性 | **工作稳定性** | 直接映射 |
| 4. 社会认可度与家人期望 | **家庭支持度** | 近似映射。考生模式下通常满分，家长模式下根据对话判断 |
| 5. 工作地点自由度 | **不参与加权** | 作为 Stage 3 的筛选条件使用：排除地域严重不匹配的专业，在报告中标注地域风险 |

### 映射示例

用户排序：兴趣(1) > 稳定(2) > 就业薪资(3) > 地点(4) > 认可(5)

- 兴趣匹配度 → 排第1，权重 × 2.0
- 工作稳定性 → 排第2，权重 × 1.5
- 就业前景 → 排第3（与薪资共享），权重 × 1.2
- 薪资水平 → 排第3（与就业共享），权重 × 1.2
- 能力匹配度 → 未上榜，保持默认权重
- 深造必要性 → 未上榜，保持默认权重
- 家庭支持度 → 排第5，保持默认权重
- 地点自由度 → 排第4，不参与加权，作为筛选条件

## 评分维度详解

### 1. 兴趣匹配度（1-5）
**依据**：Stage 1 的霍兰德类型与专业特征的重合度。

| 分数 | 含义 |
|------|------|
| 5 | 霍兰德核心类型与专业高度匹配（如研究型→基础科学） |
| 4 | 主要类型匹配，次要类型有交叉 |
| 3 | 部分匹配，但专业涉及不擅长的领域 |
| 2 | 匹配度较低，但无明显排斥 |
| 1 | 明显不匹配，用户类型与该专业特征相悖 |

**参考映射：**

| 霍兰德类型 | 高匹配专业 |
|-----------|-----------|
| 研究分析型 | 理学、基础医学、数据科学、哲学 |
| 艺术创意型 | 设计学、建筑学、文学创作、数字媒体 |
| 社会服务型 | 教育学、护理学、社会工作、旅游管理 |
| 企业管理型 | 工商管理、金融学、法学、市场营销 |
| 常规事务型 | 会计学、审计学、档案管理、行政管理 |
| 技术实操型 | 机械、电气、交通运输、医学检验、烹饪 |

### 2. 能力匹配度（1-5）
**依据**：用户的高中优势学科与该专业核心能力要求的对应关系（见 `references/majors-reference.md` 中的学科映射表）。

| 分数 | 含义 |
|------|------|
| 5 | 核心学科均为用户优势学科 |
| 4 | 核心学科大部分匹配 |
| 3 | 核心学科基本匹配，但有薄弱环节 |
| 2 | 需要用户并非优势的学科能力 |
| 1 | 核心能力要求与用户学科优势完全不符 |

### 3. 就业前景（1-5）
**依据**：web_search 获取的行业趋势、人才需求、饱和度数据。

| 分数 | 含义 |
|------|------|
| 5 | 朝阳行业，人才缺口大，政策重点扶持（绿牌） |
| 4 | 需求稳定增长，就业面广 |
| 3 | 需求平稳，竞争适中 |
| 2 | 需求趋于饱和，竞争激烈（黄牌） |
| 1 | 供过于求，大量失业或转行（红牌） |

### 4. 薪资水平（1-5）
**依据**：web_search 获取的应届起薪 + 3-5 年薪资增长空间。

| 分数 | 含义（本科） | 含义（专科） |
|------|------------|------------|
| 5 | 应届 15w+ | 应届 10w+ |
| 4 | 应届 10-15w | 应届 7-10w |
| 3 | 应届 7-10w | 应届 5-7w |
| 2 | 应届 5-7w | 应届 4-5w |
| 1 | 应届 <5w | 应届 <4w |

> 注：具体数值以 web_search 实际结果为准，上述为通用参考值。

### 5. 工作稳定性（1-5）
**依据**：考编/国企岗位数量、行业周期波动性、替代风险。

| 分数 | 含义 |
|------|------|
| 5 | 体制内岗位多（师范、医学、公安、水利），几乎无周期波动 |
| 4 | 大型国企岗位丰富（电气、石油、铁路），抗周期能力强 |
| 3 | 部分国企机会 + 稳定民企，一般性行业周期 |
| 2 | 主要靠民企和外企就业，行业有周期性 |
| 1 | 高度市场化，波动大，受政策和经济周期影响明显 |

### 6. 深造必要性（1-5，反转分）
**含义**：该专业是否必须读研才能有较好就业——分数越低越好（不需要深造 = 高性价比）。

| 分数 | 含义 |
|------|------|
| 5 | 本科/专科毕业即可好的就业（如口腔医学、护理、软件工程） |
| 4 | 读研有优势但非必须（如会计学、电气工程） |
| 3 | 读研能显著提升就业质量（如金融学、法学） |
| 2 | 读研几乎是就业门槛（如生物科学、临床医学） |
| 1 | 必须读到博士才有对口岗位（如天文学、理论物理、哲学） |

### 7. 家庭支持度（1-5）
**依据**：家长模式下加权更重，考生自评模式下通常默认满分或根据对话判断。

| 分数 | 含义 |
|------|------|
| 5 | 家人完全支持 |
| 4 | 家人有一定疑虑但可沟通 |
| 3 | 家人不太理解这个方向 |
| 2 | 家人明显不太支持 |
| 1 | 家人坚决反对 |

## 权重计算

### 默认权重（考生模式）

| 维度 | 默认权重 |
|------|----------|
| 兴趣匹配度 | 25% |
| 能力匹配度 | 20% |
| 就业前景 | 20% |
| 薪资水平 | 15% |
| 工作稳定性 | 10% |
| 深造必要性 | 5% |
| 家庭支持度 | 5% |

### 自定义权重

根据 Stage 1 用户的价值排序动态调整（参考上方「价值观→维度映射」确定每个排名的目标维度）：

- 排第 1 的维度权重 × 2.0
- 排第 2 的维度权重 × 1.5
- 排第 3 的维度权重 × 1.2
- 排第 4、第 5 保持默认

**注意**：当排第 3 是"就业前景与薪资水平"时，就业前景和薪资水平各 ×1.2。如果某个排名映射到多个维度，各维度共享该倍率。

然后**归一化到总和 100%**。

### 家长模式调整
家长模式下，「家庭支持度」权重默认提升到 15%，从其他维度等比例扣除。

---

## 加权总分公式

```
总分 = Σ (各维度评分 × 对应权重)
```

满分 = 5 分，按总分降序排列即为推荐排序。

## 风险标注规则

对每个候选专业检查以下风险标签：

- 🔴 **红牌预警**：就业率连续多年垫底、大量毕业生转行
- 🟡 **黄牌预警**：就业饱和度较高、竞争激烈
- 📚 **深造刚需**：本科就业困难，必须读研（深造必要性 ≤ 2）
- 🏥 **体检限制**：该专业对色觉、视力、身高有硬性要求
- 📍 **地域锁定**：就业机会集中在少数城市，回老家就业困难
- 🤖 **AI 替代风险**：工作内容高度重复，5-10 年内可能被 AI 取代
- 👨‍👩‍👧 **家庭分歧**：家长与考生意见不一致（需在报告中说明双方理由）
