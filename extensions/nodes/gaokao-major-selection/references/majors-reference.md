# 中国高等教育专业目录索引

## 本科：12 大学科门类

### 01 哲学
- 哲学、逻辑学、宗教学、伦理学
- **学科映射**：政治 → 哲学；语文+历史 → 哲学
- **典型就业**：高校教师、公务员、出版编辑、文博机构
- **深造提醒**：对学历要求高，建议读研

### 02 经济学
- 经济学、经济统计学、国民经济管理、金融学、金融工程、保险学、投资学、国际经济与贸易、财政学、税收学
- **学科映射**：数学+政治 → 经济学；数学 → 金融学
- **典型就业**：银行、证券、基金、咨询、公务员、企业财务
- **专科对应**：金融服务与管理、财富管理、国际经济与贸易
- **饱和度**：金融学部分方向供过于求（黄牌）

### 03 法学
- 法学、知识产权、监狱学、政治学与行政学、国际政治、外交学、社会学、社会工作、民族学、马克思主义理论、公安学类
- **学科映射**：政治+语文 → 法学；政治+历史 → 政治学
- **典型就业**：律师、公检法、公司法务、公务员、社会组织
- **深造提醒**：法学需要通过法考，建议读研
- **专科对应**：法律事务、法律文秘、社区矫正

### 04 教育学
- 教育学、教育技术学、学前教育、小学教育、特殊教育、体育教育、运动训练、社会体育指导与管理、武术与民族传统体育
- **学科映射**：各学科均可 → 对应学科师范方向；体育 → 体育教育
- **典型就业**：中小学教师、培训机构、教育管理、体育教练
- **专科对应**：学前教育、小学教育、体育教育
- **趋势**：学前教育需求稳定，中小学教师岗位竞争加大

### 05 文学
- 汉语言文学、汉语言、汉语国际教育、中国少数民族语言文学、古典文献学
- 英语、俄语、德语、法语、西班牙语、阿拉伯语、日语、朝鲜语等小语种
- 新闻学、广播电视学、广告学、传播学、编辑出版学、网络与新媒体
- **学科映射**：语文+英语 → 中国语言文学/外国语言文学；语文 → 新闻传播
- **典型就业**：教师、记者编辑、公关广告、翻译、新媒体运营、公务员
- **专科对应**：应用英语、商务日语、网络新闻与传播
- **风险提示**：翻译受 AI 冲击较大；小语种需关注对象国经济

### 06 历史学
- 历史学、世界史、考古学、文物与博物馆学、文物保护技术
- **学科映射**：历史 → 历史学；历史+地理 → 考古学
- **典型就业**：教师、文博机构、考古所、出版社、公务员
- **深造提醒**：就业面窄，建议读研或考编

### 07 理学
- 数学与应用数学、信息与计算科学、物理学、应用物理学、化学、应用化学
- 天文学、地理科学、大气科学、海洋科学、地球物理学、地质学
- 生物科学、生物技术、生态学、心理学、统计学
- **学科映射**：数学 → 数学/统计；物理 → 物理/天文；化学 → 化学/应用化学；生物 → 生物科学；地理 → 地理科学
- **典型就业**：教师、科研机构、IT数据分析、制药、环保
- **深造提醒**：纯理学多数需要读研；统计学和信息与计算科学就业较好
- **专科对应**：略（理学专科较少，多转入应用型工科专科）

**专科对应（高职 19 大类相关）：**
- 装备制造大类：机械制造及自动化、数控技术、新能源汽车技术、工业机器人技术
- 电子与信息大类：计算机应用技术、软件技术、大数据技术、人工智能技术应用
- 土木建筑大类：建筑工程技术、工程造价、建筑室内设计
- 交通运输大类：铁道工程技术、城市轨道交通运营管理、新能源汽车检测
- 能源动力与材料大类：光伏工程技术、储能材料技术

**趋势提示：**
- 🟢 绿牌方向：人工智能、集成电路、新能源、机器人、大数据
- 🟡 黄牌方向：传统土木（受房地产影响）、传统机械
- 🔴 红牌方向：部分传统采矿（地域限制大）

### 09 农学
- 农学、园艺、植物保护、种子科学与工程、茶学、动物科学、动物医学、林学、园林、水产养殖学
- **学科映射**：生物+化学 → 农学/动物科学；生物+地理 → 林学/园林
- **典型就业**：农业企业、科研院所、公务员（农业局/林业局）、宠物医院
- **专科对应**：现代农业技术、园艺技术、畜牧兽医、宠物养护与驯导
- **趋势**：智慧农业、宠物经济方向向好

### 10 医学
- 临床医学、麻醉学、医学影像学、口腔医学、预防医学、中医学、针灸推拿学、中西医临床医学、药学、中药学、护理学、医学检验技术、医学影像技术、康复治疗学
- **学科映射**：化学+生物 → 临床/药学/护理；物理+生物 → 医学影像
- **典型就业**：医院、疾控中心、药企、医疗器械公司
- **专科对应**：临床医学（专科）、口腔医学（专科）、护理、药学、医学检验技术、康复治疗技术、中医康复技术
- **重要提醒**：
  - 临床医学必须读研+规培（周期长，至少 8 年）
  - 口腔医学本科即可执业，专科也可以
  - 护理专科就业率高，本科有编制优势
  - 色盲色弱不能报大部分医学专业
- **趋势**：老龄化带动康复、护理需求；口腔、眼科持续热门

### 11 军事学（略，需单独报考）

### 12 管理学
- 管理科学、信息管理与信息系统、工程管理、工商管理、市场营销、会计学、财务管理、人力资源管理、审计学、公共管理、行政管理、劳动与社会保障、土地资源管理、物流管理、电子商务、旅游管理、会展经济与管理
- **学科映射**：数学 → 会计/审计/财务管理；政治 → 行政管理
- **典型就业**：企业各职能部门、会计师事务所、公务员、银行
- **专科对应**：大数据与会计、市场营销、电子商务、现代物流管理、旅游管理、酒店管理与数字化运营
- **风险提示**：工商管理、市场营销等"万金油"专业竞争激烈，建议搭配技能型方向

### 13 艺术学
- 美术学、绘画、雕塑、设计学类（视觉传达、环境设计、产品设计、数字媒体艺术）、音乐学、舞蹈学、戏剧影视文学、播音与主持艺术、广播电视编导
- **学科映射**：美术 → 美术/设计；音乐 → 音乐学；语文 → 戏剧影视文学
- **典型就业**：设计师、影视行业、广告公司、教师、自由职业
- **专科对应**：艺术设计、数字媒体艺术设计、音乐表演、播音与主持
- **风险提示**：纯艺术方向收入分化大；数字媒体艺术需求增长
